Exporting

Exporting for itch.io (Windows)

This needs to be done from inside the Godot editor — it can't be scripted from outside it.

  1. Open the project in Godot 4.6+.
  2. Editor → Manage Export Templates — install the export templates matching your Godot version, if you haven't already.
  3. Project → Export... → Add... → Windows Desktop.
  4. Set an export path, e.g. build/windows/DialogueGraphToolkit.exe.
  5. Click Export Project, uncheck "Export With Debug".
  6. Zip up everything Godot placed in build/windows/ (the .exe plus its .pck and any .dlls) — that zip is what you upload to itch.io.
  7. On your itch.io project page, mark the Windows zip as "This file will be played in the browser" = off, and tag the upload as a Windows executable so the itch.io app can run it directly for people using the itch.io desktop app.
No browser export (yet)

If you also want a browser-playable version, that would need to be a separate HTML5 export of just the demo scene — the editor's Load/Save dialogs rely on real filesystem access that a browser export can't provide in the same way, so the editor itself isn't a good fit for an HTML5 build as-is.
