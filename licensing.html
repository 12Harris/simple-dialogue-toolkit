Licensing — please check before publishing

This repository doesn't currently include a license for the bundled third-party assets.

The following assets are bundled but not covered by the project's own license:

  • assets/PeaberryBase.ttf (font)
  • assets/test_box.png (texture)
  • assets/person_speech_bubble.PNG (speaker icon used in the editor's graph nodes)
Before publishing publicly

Confirm you have the right to redistribute these assets — either by checking the font/texture's original license terms, or swapping them for assets you know you can redistribute.

The code itself is under the LICENSE file in the repository root.
